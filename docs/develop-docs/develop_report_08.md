## Archive 功能移除

本轮维护移除了用户态 ZIP/RAR 压缩包导入能力：前端不再展示压缩包入口，拖拽与本地导入只处理图片、视频、文件夹和 `.kgpg` 插件包；后端删除了解压处理器、Rhai 压缩包下载 API、Android provider 与对应 Tauri 插件。`.kgpg` 插件包仍保留 ZIP 内部格式，与用户导入压缩包功能无关。
