# 国际化与界面交互

## 文案

- 画册相关键位可能在 `packages/i18n` 或各视图内；新增「父级画册」「移动到…」「子画册」等词条。
- 错误提示：成环、重名、删除非空父节点等。
- 收藏夹：可嵌套、可置于其它画册下，但**无删除入口**；「直接收藏」仍落在收藏夹**浅层**（见 [00-product-decisions.md](./00-product-decisions.md) §5）。

## 交互

- 画册列表从**平面列表**改为 **树形**或可展开列表；创建/重命名时选择父级。
- **拖拽**移动画册（若做）：需调用后端 `move` API 并处理乐观更新与失败回滚。
- **Android**：`useModalBack` 等现有规范不变；树形导航可能增加层级，注意返回栈与模态层关闭顺序一致。

## 与现有页面关系

- 主入口画册列表、侧栏、画册详情工具栏等分散在 `apps/main` 下，改动点随具体组件而定，但应复用统一 Store 与路由约定。
