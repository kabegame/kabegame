Provider 过滤树功能与故事：

筛选按钮展开的是 Kabegame 的 Provider 过滤树。它不是普通的单选筛选菜单，而是一棵“壁纸仓库路径树”：用户可以沿着不同路径浏览同一批壁纸，例如全部图片、按日期、按插件来源、按图片格式、按图片尺寸、按图片名称、按壁纸顺序。Provider 像应用背后的向导，把复杂数据整理成可点击的目录，让画廊、画册、任务详情和虚拟磁盘都能用相似的路径方式查看图片。

漫画里可以把 Provider 过滤树画成从“筛选”按钮长出来的发光小树或目录地图。树枝上挂着短标签：“全部”“日期”“插件”“格式”“尺寸”“名称”。看板娘像图书管理员一样顺着树枝给壁纸分类，小乌龟壳背包精灵如果化为小乌龟助手，可以负责举路牌或盖章。故事感是“图片太多也不怕，Provider 会带路”。

Provider 过滤树视觉约束：

树要像轻量弹出层或可爱的目录地图，不能画成系统文件资源管理器。节点文字尽量短，最多保留“全部、日期、插件、格式、尺寸、名称”等关键词。如果图片模型容易乱写字，可以只画图标和空标签，后期再补文字。

图片网格具体元素：

网格卡片显示二次元壁纸缩略图。卡片角落没有任何小标签，最多是视频缩略图的右上角有一个播放按钮。选中图片时卡片边框变成粉红色。
