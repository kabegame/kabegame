Kabegame 任务与运行配置布局 prompt：

用途：

当漫画主题与爬虫任务、下载进度、失败重试、任务详情、自动运行配置、定时收集相关时使用。

任务抽屉结构：

任务抽屉是全局浮层，可以从右侧滑出。它列出正在运行和已完成的爬虫任务，每项显示插件名、任务状态、进度条、已收集数量、停止 / 删除按钮。

任务详情结构：

任务详情页复用画廊布局。顶部有返回按钮、任务名、成功 / 失败 / 删除 / 去重数量、耗时；右侧或顶部有刷新、停止任务、删除任务、加入画册、查看日志、查看参数、帮助、快速设置。下方有任务筛选控件：成功图片 / 失败图片。成功模式显示图片网格和分页器。失败模式显示失败列表，每行包含 URL、错误信息、插件标签、时间、重试和删除按钮。

运行配置页结构：

运行配置页用于保存自动任务。顶部有标题、说明和“创建”按钮。下方有标签页：我的配置、推荐配置。我的配置中有过滤工具栏：按启用状态、按插件筛选。主体是运行配置卡片列表，每张卡片显示配置名、插件、定时状态、上次运行、运行按钮、更多操作。推荐配置按插件折叠分组，每组里有推荐预设卡片和“导入”按钮。

看板娘互动建议：

看板娘可以拿着计时器监督任务进度；给失败图片贴“重试”标签；从任务详情把成功图片搬进画册；在运行配置页设置闹钟，让插件每天自动出发。小乌龟壳背包精灵如果化为小乌龟助手，可以推着进度条或拿任务日志卷轴。

避免：

不要把任务页画成普通下载器。要保留 Kabegame 的图片网格和插件收集语境。不要让失败列表变成恐怖故障画面，保持轻松可爱。
