Kabegame 画册页布局 prompt：

用途：

当漫画主题与相册整理、画册轮播、子画册、虚拟磁盘、收藏分类相关时使用。

画册列表结构：

左侧保留主导航。右侧主内容区顶部是画册页标题和操作按钮：新建画册、刷新、打开虚拟磁盘、快速设置、帮助。主体是画册卡片网格。每张画册卡片像一个带封面拼贴的相册盒子，显示画册名、图片数量和几张预览缩略图。

画册详情结构：

进入某个画册后，顶部有返回按钮、画册名、图片总数、重命名、刷新、设为壁纸轮播、删除画册、创建子画册、打开虚拟磁盘、帮助、快速设置。标题下方有面包屑导航，例如“画册 / 风景 / 夜景”。再下方是标签页：图片、子画册。图片标签页使用和画廊相似的图片网格、筛选、排序、搜索、分页。子画册标签页显示子画册卡片网格。

虚拟磁盘视觉：

虚拟磁盘可以画成一个带盘符或文件夹图标的入口，连接画册结构。Windows 可以暗示为“K:\”盘符，macOS / Linux 可以画成挂载文件夹。不要画成复杂系统设置，只要作为画册页的小功能入口即可。

看板娘互动建议：

看板娘可以把壁纸放进不同画册盒子；给画册贴上“轮播用”标签；在画册详情的“图片 / 子画册”标签页之间切换；小乌龟壳背包精灵如果化为小乌龟助手，可以推着虚拟磁盘小车，把画册树搬到桌面文件夹里。

避免：

不要把画册页画成单纯文件夹列表。要有封面拼贴和图片数量。不要忘记画册详情的标签页和面包屑。不要让虚拟磁盘成为主角，除非漫画主题就是虚拟磁盘。
