本页汇总 Kabegame 桌面端的键盘快捷键，按作用范围（全局 / 画廊 / 预览 / 任务抽屉 / 对话框）分组，供你快速查阅。

:::note
部分快捷键仅在图片网格获得焦点时生效。如果按键没反应，先在网格空白处或任一图片上点一下，把焦点交给网格再试。
:::

:::caution
macOS 上以下表格中的 `Ctrl` 对应 `Cmd`（⌘）；文本中只在组合键写法与 Windows/Linux 不同处单列一行。Android 设备通常没有物理键盘，此页面主要面向桌面用户；应用内「帮助」页的快捷键 Tab 也仅在桌面端显示。
:::

## 全局

| 组合键 | 动作 | 平台备注 |
|---|---|---|
| `F11` | 切换全屏 | 仅 Windows / Linux |
| `Ctrl + ⌘ + F` | 切换全屏 | 仅 macOS（由系统处理，非应用内绑定） |

## 画廊（网格与选择）

需要先让图片网格获得焦点。以下快捷键在紧凑模式（窄窗口 / Android）下部分禁用。

| 组合键 | 动作 | 平台备注 |
|---|---|---|
| `Ctrl` + 鼠标滚轮 | 调整网格列数 | macOS 用 `Cmd`；「固定列数」开启时不生效 |
| `Ctrl` + `+` / `-`（或 `=`） | 调整网格列数 | macOS 用 `Cmd`；「固定列数」开启时不生效 |
| `Ctrl` + `A` | 全选当前页 | macOS 用 `Cmd` |
| `Shift` + 点击 | 范围选择 | 以上次选中的图片为锚点 |
| `Ctrl` + 点击 | 切换单张选中状态 | macOS 用 `Cmd` |
| `Ctrl` + `C` | 复制当前单选图片到剪贴板 | 仅在网格恰好选中 1 张时生效；多选请走右键菜单 |
| `Delete` / `Backspace` | 删除已选图片 | 会进入删除确认流程 |

:::tip
按住 `空格` 并用鼠标拖动，可在网格容器内进行拖拽滚动（松开空格即退出该模式）。此功能为桌面端鼠标场景设计，不适用于触控。
:::

## 预览

以下快捷键仅在图片预览对话框打开时生效；焦点位于文本输入框时会被忽略。

| 组合键 | 动作 | 平台备注 |
|---|---|---|
| `←` / `→` | 上一张 / 下一张 | — |
| `Ctrl` + `C` | 复制当前图片到剪贴板 | macOS 用 `Cmd` |
| `Delete` / `Backspace` | 删除当前图片 | 会进入删除确认流程 |
| `Esc` | 关闭预览 | — |

## 任务抽屉

任务抽屉当前未提供专用键盘快捷键。打开 / 关闭抽屉请使用工具栏按钮；抽屉内的滚动与点击遵循系统默认交互。

## 对话框

| 组合键 | 动作 | 平台备注 |
|---|---|---|
| `Enter` | 在「删除确认」对话框中执行删除 | 焦点不能位于复选框上 |
| `Esc` | 关闭对话框 | Element Plus 默认行为 |

此外，画册 / 画册详情页头部的重命名输入框支持 `Enter` 确认、`Esc` 取消。
