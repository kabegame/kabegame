## 核心功能

<CardGrid stagger>
  <Card title="插件驱动收集" icon="puzzle">
    通过 Rhai 脚本插件从各类动漫图站自动爬取图片，支持一键安装来自插件商店的社区插件。
  </Card>
  <Card title="画廊与画册管理" icon="open-book">
    统一管理所有来源的图片，通过画册自由分类整理，支持批量操作与去重。
  </Card>
  <Card title="壁纸轮播" icon="laptop">
    以画册为源自动轮播更换桌面壁纸，支持原生模式与窗口模式，可设置过渡效果与轮播间隔。
  </Card>
  <Card title="跨平台" icon="seti:config">
    Windows、macOS、Linux 桌面端及 Android 均可使用，功能大体一致。
  </Card>
</CardGrid>
