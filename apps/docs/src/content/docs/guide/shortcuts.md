:::note
部分快捷键仅在图片网格获得焦点时生效（先在网格空白处/图片上点一下）。
:::

## 全局

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F11`（Windows/Linux）/ `Ctrl+Command+F`（macOS） | 切换全屏 | 切换应用的全屏显示模式 |

---

## 网格布局

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd` + 鼠标滚轮 | 调整网格列数 | 按住 Ctrl（macOS 为 Cmd）并滚动鼠标滚轮，可快速调整图片网格的列数 |
| `Ctrl/Cmd` + `+` / `-`（或 `=`） | 调整网格列数 | 按住 Ctrl（macOS 为 Cmd）并按 +/-（或 =），可调整图片网格的列数 |

---

## 选择/删除

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd` + `A` | 全选 | 在图片网格中快速全选当前页面的所有图片 |
| `Shift` + 点击 | 范围选择 | 在网格中按住 Shift 点击图片，可按上次选择位置进行范围选择 |
| `Ctrl/Cmd` + 点击 | 多选/取消选择 | 在网格中按住 Ctrl（macOS 为 Cmd）点击图片，可切换该图片的选择状态 |
| `Esc` | 清空选择 | 清空已选择的图片，并关闭可能打开的右键菜单 |
| `Delete` / `Backspace` | 删除选中图片 | 在图片网格中删除当前选中的图片（会进入应用的删除流程/确认） |

---

## 预览

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `←` `→` | 上一张/下一张 | 在图片预览对话框中切换上一张/下一张 |
| `Ctrl/Cmd` + `C` | 复制图片 | 在图片预览对话框中，或图片网格单选时，复制当前图片到剪贴板 |
| `Delete` / `Backspace` | 预览中删除 | 在图片预览对话框中快速删除当前图片（会进入应用的删除流程/确认） |
