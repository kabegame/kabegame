## 任务是什么？

任务是 Kabegame 中执行图片收集的工作单元。每个任务代表一次完整的图片收集过程，包括：

1. **执行插件脚本**：任务会执行某个源插件的脚本，从目标网站或本地文件系统收集图片
2. **按照配置的参数**：根据你提供的参数（如搜索关键词、页码、URL 等）执行收集逻辑
3. **下载图片到指定位置**：将收集到的图片下载并保存到你指定的输出目录
4. **添加到画廊和画册**：下载完成后，图片会自动添加到画廊，如果指定了画册，也会添加到对应画册

任务创建后，会在后台自动执行，你可以在任务抽屉中查看进度和状态。

### 最大运行任务数量

Kabegame 限制同时运行的任务数量为 10 个，超过 10 个的任务会进入「等待中」状态，排队等待执行。

任务的执行速度受**最大并发下载量**影响（可在「设置」→「下载设置」中配置，范围 1–10）。该设置控制所有任务总共同时下载的图片数量，而不是单个任务的并发数。

例如：如果设置为 5，那么即使有 10 个任务在运行，同时最多只会下载 5 张图片。

:::tip
如果任务一直显示「等待中」，可能是因为达到了最大并发下载数限制。你可以：
- 等待其他任务完成，释放下载槽位
- 在设置中增加「最大并发下载量」（注意：过高的并发数可能影响网络稳定性）
- 手动停止或删除不需要的任务
:::

## 查看任务信息与状态

### 任务抽屉

在画廊、画册的右上角有一个任务按钮，点击后可以打开任务抽屉，查看所有任务的列表。

任务抽屉中会显示：

- **任务名称**：显示任务使用的插件名称和输出目录
- **任务状态**：等待中、运行中、已完成、失败、已取消
- **进度信息**：显示已收集的图片数量和总进度
- **操作按钮**：查看详情、停止任务、删除任务等

### 任务详情页

在任务抽屉中点击任务，或右键任务选择「查看详情」，可以进入任务详情页。

任务详情页包含以下信息：

| 信息 | 说明 |
|------|------|
| 基本信息 | 任务名称、状态、使用的插件、输出目录等 |
| 已收集的图片 | 以网格形式展示任务已收集的所有图片 |
| 失败记录 | 如果有图片下载失败，显示失败原因和重试选项 |
| 操作按钮 | 刷新、停止任务、删除任务等 |

### 任务状态说明

| 状态 | 说明 |
|------|------|
| 等待中 | 任务已创建，等待开始执行 |
| 运行中 | 任务正在执行，正在收集图片 |
| 已完成 | 任务已成功完成，所有图片都已收集 |
| 失败 | 任务执行过程中出现错误，已停止 |
| 已取消 | 任务被用户手动取消 |

### 查看任务图片

在任务详情页中，所有已收集的图片会以网格形式展示，你可以：

- 滚动查看任务收集的所有图片
- 双击图片打开预览对话框，查看大图
- 点击图片进行选择，支持多选和范围选择
- 选择图片后右键菜单可以「加入画册」
- 选择图片后可以删除任务中的图片（不影响已下载的文件）

## 创建任务

**方式一：通过收集对话框创建**

在画廊页面点击右上角的「收集」按钮，打开收集对话框：

1. 选择源：从下拉列表中选择要使用的插件
2. 配置参数：根据插件要求填写必要的参数
3. 输出目录：选择图片保存的位置（留空使用默认下载目录）
4. 输出画册：可选，选择将图片添加到哪个画册（也可以新建画册）
5. 运行配置：可选，选择之前保存的运行配置快速复用参数
6. 点击「开始收集」按钮即可创建任务

**方式二：拖入文件自动创建导入任务**

将本地图片文件、文件夹或压缩包（zip）直接拖入应用窗口，会自动创建本地导入任务。

## 终止任务

如果任务正在运行，你可以随时终止它：

- **任务抽屉**：打开任务抽屉，找到要终止的任务，右键点击选择「停止」，确认后任务将被终止
- **任务详情页**：进入任务详情页，如果任务状态为「运行中」，右上角会显示「停止任务」按钮，点击即可终止任务

终止任务后，已下载的图片会保留，任务状态会变为「已取消」。

---

## 删除任务

- **任务抽屉**：打开任务抽屉，找到要删除的任务，右键点击选择「删除」
- **任务详情页**：进入任务详情页，点击右上角的「删除任务」按钮
- **批量操作**：在任务抽屉底部点击「清除所有已完成/失败/已取消的任务」，可以批量删除这些任务（等待中和运行中的任务会被保留）

:::note
如果任务正在运行，删除前会自动先终止任务，然后再删除。删除任务只会删除任务记录，不会删除已下载的图片文件。
:::
