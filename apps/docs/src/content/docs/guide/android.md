Kabegame 的 Android 版与桌面版共享同一套界面，但受平台限制做了不少裁剪，也加进了几个只有移动端才有的入口。本页以「与桌面版的差异」为骨架，帮你快速理清手机上能做什么、不能做什么。

## Android 独有

### 通过系统媒体选择器导入图片

Android 不支持拖拽导入，只保留一个入口：在画廊右上角点击「开始收集」→ 选择「本地」，系统会弹出 Photo Picker，勾选要导入的图片后返回即可入库。选择文件夹时走系统目录选择器（SAF），应用会持久持有授权，目录内容不会被复制到私有空间。

### 安装 `.kgpg` 插件包

在插件管理页可以通过系统文件选择器选取 `.kgpg` 文件进行本地安装，无需连接插件商店。

### 分享到其他 App

长按图片调出 ActionSheet，在「更多」子菜单中点击「分享」，会触发系统分享面板，可把图片交给任意外部 App。桌面版的「复制图片」在 Android 上等价于将图片 URI 写入系统剪贴板，不同 ROM 对粘贴图片 URI 的支持程度不一。

### 设为壁纸

长按图片 → 「抱到桌面上」，应用调用系统 `WallpaperManager` 写入壁纸，成功后提示「壁纸设置成功」。

:::note
主屏与锁屏的覆盖行为取决于系统版本，Android 并未向应用暴露稳定的分屏/锁屏选项，请以系统实际表现为准。
:::

壁纸显示方式在 Android 上固定为「填充」，桌面端的其他样式不可选。

### 壁纸轮播（后台调度）

在「设置 → 壁纸设置」中启用「启用壁纸轮播」，Android 会使用系统 `WorkManager` 作为周期调度器。

| 设置项 | 说明 |
|---|---|
| 启用壁纸轮播 | 开启后由 WorkManager 调度，关闭后取消调度。 |
| 轮播间隔 | Android 最小 15 分钟（系统硬限制，低于此值自动修正）。 |
| 轮播画册 | 留空 = 全画廊；指定画册 = 仅该画册参与轮播。 |
| 包含子画册 | 开启后子画册图片一并纳入轮播池。 |
| 轮播模式 | 随机 / 顺序。 |

:::caution
`WorkManager` 周期任务受 Android 电池优化策略影响。若应用未获得「后台运行」豁免，轮播可能被系统暂停。详见下方「电池优化豁免引导」一节。
:::

### 后台任务前台服务通知

下载任务开始后，应用会显示一条常驻前台通知，用于提升存活率防止被系统回收；全部任务完成后通知会切换成可点击状态，点击可重新打开应用。Android 13+ 首次使用会请求通知权限。

:::note
当前前台通知文本暂以中文硬编码（例如「任务进度」「任务已全部完成」「点击打开应用」），尚未接入多语言。
:::

### 电池优化豁免引导

Android 的省电策略会冻结后台应用，导致下载停滞或轮播失效。应用会在以下时机自动提示你授予豁免：

- 首次打开任务抽屉；
- 首次开启壁纸轮播。

此外，任务抽屉顶部和壁纸轮播开关旁会显示一枚闪电图标，悬浮提示「后台运行受限，点击开启」，方便你随时去开权限。

### 返回键按栈关闭弹层

按一次 Android 系统返回键，Kabegame 会先关闭当前最顶层的弹层（对话框、抽屉、ActionSheet、图片预览、选择模式等）；只有全部弹层都关完，返回键才会交回给路由栈或系统处理。这让你在深层菜单里也不会一下退出到桌面。

## Android 不支持

以下桌面端能力在 Android 上不存在，对应入口会直接隐藏：

| 能力 | 说明 |
|---|---|
| 虚拟盘 / 画册盘 | Dokan / macFUSE / FUSE 不在 Android 编译范围内，挂盘、「在资源管理器打开盘」等入口都不会出现。 |
| 系统托盘 | 移动端没有托盘概念，桌面版的「下一张壁纸 / 显示 / 退出」菜单项不存在。 |
| 开机自启 / 全局快捷键 | 平台不允许，相关设置项隐藏。 |
| CLI（`kabegame-cli`） | 不随 APK 打包。 |
| MCP 服务器 / HTTP 文件服务 / IPC 服务 | 全部在 Android 构建中被禁用。 |
| Wallpaper Engine 导出 | Wallpaper Engine 无 Android 版，菜单项仅 Windows 可见。 |
| 拖拽导入（文件/文件夹/zip） | 只能通过「开始收集 → 本地」的系统选择器导入。 |
| 打开所在文件夹 | Android 没有文件管理器定位的通用机制，菜单项隐藏。 |
| WebView 相关设置 | WebView 入口、自动打开、清理数据、开发者 WebView 等选项全部隐藏。 |
| 快捷键帮助页 | 无物理键盘前提下被隐藏。 |
| 画廊列数 / 对齐方式 / 图片宽高比 / 默认下载目录 | Android 使用固定布局或不同的选择组件，桌面的对应设置项不展示。 |

## 排障

**后台下载 / 壁纸轮播被系统停止** → 电池优化仍开启 → 点击任务抽屉顶部或壁纸开关旁的闪电图标，按引导豁免后台运行权限。

**壁纸轮播间隔实际仍为 15 分钟** → Android `WorkManager` 对周期任务有 15 分钟系统硬限制，低于此值的配置会自动修正。

**收不到任务完成通知（Android 13+）** → 通知权限未授予 → 前往系统「设置 → 应用 → Kabegame → 通知」打开通知权限。

**返回键没有关闭弹层，直接退出了应用** → 该弹层可能未注册到返回键栈，属于应用问题，可通过反馈渠道提交，附上操作路径。
