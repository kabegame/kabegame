---
title: 壁纸
description: 设置桌面壁纸、轮播和显示模式，支持视频壁纸、过渡效果与 Wallpaper Engine 导出。
---

Kabegame 的壁纸系统围绕三个维度组合：**来源**（单张 / 轮播）、**显示后端**（原生 / 窗口 / Plasma 插件 / Android 系统）、**外部集成**（Wallpaper Engine 导出，仅 Windows）。本页覆盖每种组合在不同平台上的可用能力与注意事项。

## 单张壁纸

在画廊、画册或任务详情中，右键点击任意图片（Android 长按），选择「抱到桌面上」，即可将该图片设为桌面壁纸。这是最快捷的设置方式，无需进入设置页。

**其他入口：**

- **设置页**：进入「设置」→「壁纸轮播」标签，顶部的「当前壁纸：」会显示当前壁纸缩略图与路径。
- **画册详情页**：点击右上角「设为轮播壁纸」按钮——**这不是单张**，而是把该画册作为轮播来源，并开启轮播。

:::note
设置单张壁纸**不会**自动关闭轮播。若轮播正在运行，下一次间隔到期时仍会把这张壁纸换掉。如需固定某张图片，请手动在设置页关闭「启用壁纸轮播」。
:::

若你选择的是视频，而当前处于原生模式，应用会弹出确认：「该媒体需要窗口模式才能设置为壁纸。是否切换到窗口模式？」确认后会自动切换模式再应用。

## 壁纸轮播

轮播会按固定间隔从画册或整个画廊中挑选下一张壁纸，经当前的显示后端呈现。

### 启用与来源

- **开启**：「设置」→「壁纸轮播」→ 开启「启用壁纸轮播」。
- **来源**：「选择画册」项目可选具体画册，或「全画廊」（从整个画廊抽样；顺序模式下从当前壁纸开始继续）。
- **包含子画册**：开启「轮播包含子画册」会把所选画册及其子画册的图片合并后参与轮播。
- **画册详情页快捷**：在画册顶部点「设为轮播壁纸」会同时开启轮播并把该画册设为来源。

### 间隔与模式

| 选项 | 取值 | 说明 |
|---|---|---|
| 轮播间隔 | 桌面 1–1440 分钟；Android 最小 15 分钟 | 改动立即生效 |
| 轮播模式 | 随机 / 顺序 | 随机会尽量避免连续两张相同；顺序从当前壁纸继续 |

### 降级规则

- **所选画册为空或被删除** → 自动回退到全画廊取图。
- **画廊也为空** → 关闭轮播并清除「当前壁纸」。
- **原生模式下遇到视频** → 自动跳过该视频；只有图片会被采用。

## 显示模式

「壁纸模式」决定壁纸如何呈现在桌面上。切换时会弹出确认，应用会先停止轮播，重置当前壁纸（若文件不存在则从画廊挑一张兜底），再恢复轮播。

### 原生模式

直接调用系统壁纸 API。资源占用低、应用退出后壁纸由系统保留，但过渡与显示方式受系统限制。

- **Windows**：支持 fill / fit / stretch / center / tile 以及「跟随系统」；过渡只有「跟随系统」（Windows 会触发系统级淡入）。
- **macOS**：显示方式**仅支持「跟随系统」**，样式由系统决定；无自定义过渡。
- **Linux GNOME / 其他桌面**：支持全部显示方式；无自定义过渡。
- **Linux KDE Plasma**：显示方式支持 fill / fit / center / tile（不包含 stretch）；无自定义过渡。

### 窗口模式（仅 Windows / macOS）

在桌面层上创建一个常驻透明窗口来渲染壁纸，支持视频与所有过渡效果。代价是需要保持应用运行：退出后窗口销毁，桌面回到系统壁纸。

- 显示方式：fill / fit / stretch / center / tile 全部可用。
- 过渡效果：无 / 淡入淡出 / 滑动 / 缩放。
- 视频：支持 mp4、mov（见下文「视频壁纸」）。

:::caution
macOS 下的窗口模式视频为**实验性**支持。若出现黑屏或播放异常，改回原生模式或切换视频格式。
:::

### Plasma 插件模式（仅 Linux + KDE Plasma）

把系统壁纸插件替换为 Kabegame Plasma 插件来接管渲染，能力接近窗口模式（支持过渡、视频、全部显示方式），但无需独立透明窗口。

- 此选项**仅在检测到已安装 Kabegame Plasma 插件时**出现在「壁纸模式」下拉里。
- 切换时会提示：「切换到插件模式将修改系统壁纸插件为 Kabegame 壁纸插件，是否继续？」

:::note
KDE Plasma 集成步骤参见后续专题。
:::

### Android

Android 没有壁纸模式选择，统一走系统 `WallpaperManager`：

- 显示方式**只能「填充」**，无自定义过渡。
- **主屏 / 锁屏**：Kabegame 不显式区分锁屏，最终由系统决定该壁纸作用到哪一层。
- 应用退出后，系统壁纸已永久写入并保留，但轮播不再推进。
- Android 的返回栈、图片选择器等交互详见 [Android 专版说明](/guide/android/)。

## 显示方式（Style）

「壁纸显示方式」控制图片如何贴合屏幕：

| 取值 | 含义 |
|---|---|
| 填充（fill） | 裁剪至覆盖屏幕 |
| 适应（fit） | 整图可见，留边 |
| 拉伸（stretch） | 强制拉伸填满（可能变形） |
| 居中（center） | 原尺寸居中 |
| 平铺（tile） | 小图重复铺满 |
| 跟随系统（system） | 由系统当前壁纸样式决定 |

可用集取决于模式与平台，见下文「平台差异」一节。

## 过渡效果

「过渡效果」仅在轮播切换时体现。未开启轮播时，设置页会提示「未启用轮播：过渡效果将在启用轮播后生效」。

| 模式 | 可用过渡 |
|---|---|
| 原生（Windows） | 跟随系统（系统级淡入） |
| 原生（macOS / Linux） | 跟随系统 |
| 窗口 / Plasma 插件 | 无 / 淡入淡出 / 滑动 / 缩放 |
| Android | 不适用 |

图片与视频在窗口 / 插件模式下共用同一套过渡动画。

## 视频壁纸

- **支持格式**：mp4、mov。GIF 按图片处理，原生模式即可。
- **模式要求**：视频必须在**窗口模式**（Windows / macOS）或 **Plasma 插件模式**下才能设置；在原生模式下右键视频会触发「切换窗口模式」确认。
- **音量 / 播放速率**：在窗口模式或 Plasma 插件模式下，设置页会出现「视频壁纸音量」（0–1）与「视频播放速率」（0.25–3）。修改后实时生效。
- **轮播时**：原生模式自动跳过视频；窗口 / 插件模式下图片与视频都参与轮播。

:::note
前端只按扩展名判断是否为视频。mp4 / mov 内的编码若系统播放器不支持，可能出现黑屏——此时换一个文件或重新压制。
:::

## 导出到 Wallpaper Engine（仅 Windows）

Kabegame 可以把图片或视频导出为 **Wallpaper Engine Web 工程**（包含 `project.json`、`index.html`、资源文件），写入 WE 的 `projects/myprojects` 目录后由 WE 识别。

:::note
这不是订阅或发布到 WE Workshop，也不会自动在 WE 内「应用」工程。导出完成后你仍需打开 WE，在「我的项目」里手动选择该工程。
:::

**使用步骤：**

1. 首次使用前，前往「设置」→「壁纸轮播」→「Wallpaper Engine 目录」，选择 WE 安装目录或直接选到 `projects/myprojects` 任一层级。未配置时会 toast「未配置 Wallpaper Engine 目录：请到 设置 -> 壁纸轮播 -> Wallpaper Engine 目录 先选择」。
2. 右键目标图片或视频 → 「更多」→「导出到 Wallpaper Engine」。
3. 在弹出的输入框里填工程名（留空使用默认名称）。
4. 导出完成后会 toast 成功信息并自动打开工程目录。切换到 WE，从「我的项目」里选中即可。

**限制：**

- 从 WE Workshop 拉取 / 订阅工程：不支持。
- 在 WE 里直接「应用」工程：需用户手动选。
- 视频工程：仅接受 mp4 / mov；其他扩展名会被拒绝。
- macOS / Linux / Android：右键菜单不会出现该选项。

## 应用关闭后的壁纸

| 场景 | 退出后行为 |
|---|---|
| Windows 原生 | 系统已记录，继续显示 |
| Windows 窗口模式 | 透明窗口销毁，恢复系统壁纸 |
| macOS 原生 | 系统设置已写入，保留 |
| macOS 窗口模式 | 透明窗口销毁，恢复系统壁纸 |
| Linux 原生 | 由桌面环境持久化，保留 |
| Linux Plasma 插件 | plasmoid 由 Plasma Shell 管理；具体行为取决于插件实现 |
| Android | 系统壁纸已永久写入并保留；轮播不再推进 |

想让壁纸持续轮播，请让 Kabegame 在后台运行（最小化到托盘，不要选「退出」）。

## 平台差异

| 能力 | Windows | macOS | Linux（GNOME / 其他） | Linux（Plasma） | Linux（Plasma + 插件） | Android |
|---|---|---|---|---|---|---|
| 原生模式 | 是 | 是（样式跟随系统） | 是 | 是（无 stretch） | 是 | — |
| 窗口模式 | 是 | 是（视频实验性） | 否 | 否 | 否 | 否 |
| Plasma 插件模式 | 否 | 否 | 否 | 否 | 是 | 否 |
| 单张图片壁纸 | 是 | 是 | 是 | 是 | 是 | 是（仅填充） |
| 单张视频壁纸 | 需窗口模式 | 需窗口模式 | 否 | 否 | 需插件模式 | 否 |
| 显示方式 | 全部 | 原生仅 system；窗口全部 | 全部 | fill/fit/center/tile | 全部 | 仅 fill |
| 自定义过渡（fade/slide/zoom） | 仅窗口模式 | 仅窗口模式 | 否 | 否 | 插件模式 | 否 |
| 视频音量 / 播放速率 UI | 是 | 是 | 否 | 是 | 是 | 否 |
| 轮播 | 是 | 是 | 是 | 是 | 是 | 是（最小 15 分钟） |
| 轮播包含子画册 | 是 | 是 | 是 | 是 | 是 | 是 |
| Wallpaper Engine 导出 | 是 | 否 | 否 | 否 | 否 | 否 |
| 主屏 / 锁屏分离 | 否 | 否 | 否 | 否 | 否 | 否（由系统决定） |
