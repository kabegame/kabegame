## 基本壁纸设置

在画廊、画册或任务详情中，右键点击任意图片，选择「抱到桌面上」，即可将该图片设置为桌面壁纸。这是设置单张壁纸最快捷的方式，无需进入设置页面。

**其他设置方式：**

- **在设置页选择**：进入「设置」→「壁纸轮播」标签，在「选择壁纸」中选择单张图片（轮播关闭时）
- **在画册详情页设置**：打开画册详情页，点击右上角的「设为轮播壁纸」按钮（会开启轮播模式）

**注意事项：**

- 设置壁纸后，图片会立即应用到桌面
- 如果当前启用了壁纸轮播，设置单张壁纸会自动关闭轮播，切换到单张壁纸模式
- 壁纸设置会立即生效，无需重启应用

## 轮播 vs 非轮播

Kabegame 支持两种壁纸模式：**轮播模式**和**非轮播模式**（单张壁纸）。

### 非轮播模式（单张壁纸）

- 显示一张固定的壁纸
- 需要手动更换壁纸（右键图片选择「抱到桌面上」）
- 适合：想要固定某张图片作为壁纸的场景

### 轮播模式

- 从指定画册中自动轮播更换壁纸
- 可以设置轮播间隔（分钟）和轮播模式（随机/顺序）
- 支持过渡效果（淡入淡出、滑动等）
- 适合：想要桌面壁纸自动更换的场景

### 如何开启/关闭轮播

- **开启轮播**：进入「设置」→「壁纸轮播」标签，开启「启用壁纸轮播」，然后选择要轮播的画册
- **关闭轮播**：在设置页关闭「启用壁纸轮播」，或右键图片设置单张壁纸（会自动关闭轮播）
- **在画册详情页开启**：打开画册详情页，点击「设为轮播壁纸」按钮，会自动开启轮播并选择该画册

### 轮播设置选项

| 选项 | 说明 |
|------|------|
| 轮播间隔 | 设置壁纸更换的间隔时间（1–1440 分钟） |
| 轮播模式 | 随机（每次从画册中随机选择）或顺序（按画册中的顺序依次更换） |
| 过渡效果 | 轮播切换时的动画效果（仅轮播模式支持；图片与视频均支持同一套过渡） |

**注意事项：**

- 轮播模式需要选择一个画册作为图片源，画册为空时无法轮播
- 如果删除的画册正在被轮播引用，轮播会自动关闭，切回单张壁纸模式
- 设置单张壁纸（右键图片）会自动关闭轮播，切换到非轮播模式

## 原生模式 vs 窗口模式

Kabegame 提供两种壁纸显示模式：**原生模式**和**窗口模式**，各有优缺点。

### 原生模式

- **性能更好**：直接使用系统 API 设置壁纸，资源占用低
- **功能有限**：受系统限制，过渡效果较少（仅支持无过渡和淡入淡出）
- **兼容性好**：所有系统都支持
- **壁纸显示方式**：根据系统支持显示可用样式（填充、适应、拉伸等）

### 窗口模式

- **功能更灵活**：类似 Wallpaper Engine，支持所有过渡效果
- **性能稍差**：需要创建一个透明窗口覆盖桌面，资源占用稍高
- **平台支持**：Windows / macOS 桌面端支持窗口模式；Android 使用系统壁纸 API
- **壁纸显示方式**：支持所有显示方式（填充、适应、拉伸、居中、平铺等）

### 如何切换模式

进入「设置」→「壁纸轮播」标签，找到「壁纸模式」选项，选择「原生模式」或「窗口模式」。切换模式后，当前壁纸会立即重新应用，无需重启应用。

### 过渡效果的区别

| 模式 | 可用过渡效果 |
|------|------|
| 原生模式 | 无过渡、淡入淡出 |
| 窗口模式 | 无过渡、淡入淡出、滑动、缩放等（图片与视频均支持） |

**选择建议：**

- 追求性能 → 原生模式，资源占用低，适合大部分电脑
- 追求效果 → 窗口模式，支持更多过渡效果，体验更丰富
- 仅使用单张壁纸 → 两种模式差异不大，原生模式更省资源
- 使用轮播 → 窗口模式可以体验更多过渡效果

## 视频壁纸与应用关闭说明

### 视频/动图壁纸（仅窗口模式）

在窗口模式下，除了静态图片外，还可以将视频或 GIF 动图设为桌面壁纸，壁纸会以动态方式在桌面下层全屏播放。

- **Windows**：窗口模式支持将 MP4 视频设为壁纸；GIF 等可走原生模式
- **macOS**：窗口模式支持视频和 GIF，均通过桌面下层窗口播放

若当前为原生模式且你选择的媒体需要窗口模式（如视频），应用会提示先切换到「窗口模式」再设置壁纸；也可在「设置」→「壁纸轮播」中先改为窗口模式，再设置该媒体为壁纸。

### 应用关闭后壁纸会消失

无论是原生模式还是窗口模式，壁纸都是由 Kabegame 应用提供或渲染的。当你完全退出应用（包括从托盘退出、关闭主窗口并退出等）后，桌面壁纸会恢复为系统原本的壁纸或空白，这是正常现象。

**原因：** 窗口模式下壁纸由应用内的透明窗口渲染；原生模式下虽调用了系统 API，但部分系统在应用退出后也会恢复默认。

**若想保持壁纸：** 请保持 Kabegame 在后台运行（如最小化到托盘、不选择「退出」），壁纸会持续显示。
