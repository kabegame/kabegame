设置页按功能分成四个 Tab：「应用设置」「壁纸设置」「下载设置」「插件默认」。你上次停留的 Tab 会被记住，下次打开设置时自动恢复。修改会即时保存；如果保存失败，页面上的值会回滚并提示错误。

:::note
部分在后台执行的提示文案（主要集中在壁纸轮播开/关的成功与错误反馈）目前仅有中文版本，其它语言会暂时显示中文，这不影响功能。
:::

## 应用设置

### 语言

- UI 标签：「语言」
- 作用：切换界面显示语言，支持简体中文、英文、日文、韩文等。
- 默认值：跟随系统。

### 开机启动（仅桌面）

- UI 标签：「开机启动」
- 作用：系统启动时自动运行 Kabegame。
- 默认值：关闭。

### 画册盘（仅桌面，Light 模式不可见）

- UI 标签：「画册盘」
- 作用：以虚拟盘的形式在系统文件管理器中浏览画册（仅支持只读类操作）。开启前必须先在输入框填写挂载点（例如 `K:\`），否则会提示「请先填写挂载点（例如 K:\）」。开启成功后会显示「画册盘已开启」，旁边的「打开」按钮可在文件管理器中打开该盘。
- 平台实现：Windows 基于 Dokan，macOS 基于 macFUSE，Linux 基于 FUSE。

### 图片点击行为

- UI 标签：「图片点击行为」
- 作用：设置左键点击图片的行为，可选「应用内预览」或「系统默认打开」。
- 默认值：应用内预览。
- 平台：安卓固定为「应用内预览」，不显示此项。

相关：参见 [画廊 — 应用内预览](/guide/gallery/)。

### 图片宽高比（非 Android）

- UI 标签：「图片宽高比」
- 作用：影响画廊 / 画册中图片卡片的展示宽高比。下拉列表包含 16:9、16:10、21:9、4:3、5:4、3:2、32:9，以及与当前桌面分辨率匹配的项（带「（您的桌面）」标注）；若没有匹配项，会追加一个自定义项。
- 默认值：根据桌面分辨率自动初始化。
- 平台：安卓固定使用当前屏幕分辨率对应的比例，不显示此项。

### 画廊列数（非 Android）

- UI 标签：「画廊列数」／子项「固定列数」
- 作用：关闭「固定列数」时为动态列数，可用快捷键调整；开启后在 1–6 之间固定，快捷键不再生效。
- 默认值：动态列数（对应 `galleryGridColumns = 0`）。

### 每页显示数量

- UI 标签：「每页显示数量」
- 作用：画廊列表每页加载的图片数量，可选 100 / 500 / 1000。
- 默认值：100。

### 画廊布局

- UI 标签：「画廊布局」
- 作用：选择「网格」或「画廊」两种布局。「网格」使用固定宽高比格子；「画廊」将图片纵向堆叠成多列，不留空白。
- 默认值：网格。

### 图片对齐方式（非 Android）

- UI 标签：「图片对齐方式」
- 作用：图片溢出方框时的对齐位置，可选「居中」「靠上」「靠下」。
- 默认值：居中。

### 清理应用数据（仅桌面）

- UI 标签：「清理应用数据」，按钮「清理所有用户数据」
- 作用：删除所有图片、画册、任务、设置、插件配置等用户数据。操作需要经过两次确认，执行完成后应用会自动重启。

:::caution
此操作不可撤销，会永久清除全部用户数据。执行前请确认已做好备份。
:::

### 自动打开 WebView（仅桌面）

- UI 标签：「自动打开 WebView」
- 作用：启动 WebView 类爬虫插件时自动显示并聚焦 WebView 窗口。
- 默认值：关闭。

## 壁纸设置

此 Tab 在 Web 模式下隐藏。所有选项集中在「壁纸轮播设置」卡片中。

### 启用壁纸轮播

- UI 标签：「启用壁纸轮播」
- 作用：从指定画册或整个画廊中自动轮播更换桌面壁纸。画廊或画册为空时开启会失败并提示原因。
- 默认值：关闭。
- 安卓：若系统开启了省电优化，应用会提示你去放行，避免轮播被后台限制。

### 选择画册

- UI 标签：「选择画册」
- 作用：选择轮播来源。首项「全部画廊」表示从全画廊随机抽取；也可以选择某一具体画册。单张壁纸的设置请在画廊页操作。

### 轮播包含子画册

- UI 标签：「轮播包含子画册」
- 作用：开启后，所选画册的子画册图片也会参与轮播；重复图片按合并顺序保留先出现的一条。
- 默认值：开启。
- 显示条件：仅当轮播来源选择了具体画册时出现。

### 当前壁纸

- UI 标签：「当前壁纸：…」
- 作用：只读行，显示当前生效的壁纸路径；点击可在系统文件管理器中定位该文件。仅当后端返回了有效路径时才会显示。

### 轮播间隔

- UI 标签：「轮播间隔」
- 作用：两次更换壁纸之间的时间（分钟）。
- 取值范围：桌面 1–1440；安卓 15–1440（受系统后台调度限制）。
- 默认值：30 分钟。

### 轮播模式

- UI 标签：「轮播模式」
- 作用：「随机」每次随机选择，「顺序」按顺序依次更换。
- 默认值：随机。

### 壁纸显示方式

- UI 标签：「壁纸显示方式」
- 作用：选择「按照系统」「填充」「适应」「拉伸」「居中」「平铺」。可用选项取决于当前壁纸模式与平台：
  - 原生模式：Windows 支持全部；macOS 由系统接管，此项不提供选择；Linux KDE Plasma 可选填充 / 适应 / 居中 / 平铺；其它 Linux 支持全部。
  - 窗口模式 / Plasma 插件模式：支持全部五种。

### 过渡效果

- UI 标签：「过渡效果」
- 作用：可选「无过渡」「淡入淡出」「滑动」「缩放」，或「跟随系统」。原生模式下仅有「跟随系统」，其余值会被自动纠正为无过渡。
- 适用范围：窗口模式（Windows / macOS）及 KDE Plasma 插件模式。
- 提示：若当前未启用轮播，修改过渡效果会出现提示「未启用轮播：过渡效果将在启用轮播后生效」，不影响保存。

### 视频壁纸音量

- UI 标签：「视频壁纸音量」
- 作用：视频壁纸播放时的音量（0–1），与预览内的音量逻辑一致。
- 默认值：0.5。
- 平台：Windows / macOS / Linux（仅 Plasma）。

### 视频播放速率

- UI 标签：「视频播放速率」
- 作用：视频壁纸的播放速率（0.25–3，步长 0.25）。
- 默认值：1。
- 平台：Windows / macOS / Linux（仅 Plasma）。

### 壁纸模式

- UI 标签：「壁纸模式」
- 作用：选择壁纸的实现方式。
  - 「原生模式」：调用系统壁纸接口（全平台可用）。切换时会提示「切换到原生模式会覆盖系统当前壁纸设置，是否继续？」。
  - 「窗口模式」：以独立窗口显示壁纸，支持过渡效果和视频壁纸（仅 Windows / macOS）。
  - 「插件模式」：使用 KDE Plasma 壁纸插件（仅在检测到 Plasma 环境时可用）。切换时会提示「切换到插件模式将修改系统壁纸插件为 Kabegame 壁纸插件，是否继续？」。
- 切换中页面会暂时禁用相关控件，鼠标变为等待状态；若 30 秒内后端无响应会提示超时。

### Wallpaper Engine 目录（仅 Windows）

- UI 标签：「Wallpaper Engine 目录」
- 作用：填写 Wallpaper Engine 的目录，用于「导出并自动导入到 WE」。自动导入会写入其中的 `projects\myprojects` 子目录；当路径被正确识别时会以链接形式展示。

## 下载设置

| UI 标签 | 作用 | 取值范围 | 默认值 |
|---|---|---|---|
| 同时运行任务数 | 同时执行的爬虫任务数量，若遇到卡顿可调低 | 1–10 | 后端默认 |
| 最大并发下载量 | 同时下载的图片数量 | 1–10 | 后端默认 |
| 下载间隔时间 | 每次下载完成后进入下一轮前等待（毫秒） | 100–10000 | 500 |
| 网络失效重试次数 | 下载图片遇到网络错误 / 超时时的额外重试次数 | 0–10 | 后端默认 |
| 自动去重 | 根据文件哈希自动跳过重复图片 | 开关 | 后端默认 |
| 默认下载目录 | 任务未指定输出目录时的保存位置（按插件分文件夹） | 路径 | 无 |

「默认下载目录」仅在桌面显示，安卓不提供此项；支持输入路径、通过「选择」按钮挑选目录以及「恢复默认」。

相关：参见 [下载与任务](/guide/tasks/)。

## 插件默认

卡片标题「爬虫插件默认配置」。

### 导入推荐配置时默认启用定时

- UI 标签：「导入推荐配置时默认启用定时」
- 作用：从插件包导入推荐运行配置时，是否默认启用定时。关闭后，导入的配置需要在列表里手动开启定时。
- 默认值：开启。

### 爬虫插件默认配置面板

可为每个已安装插件预设默认的变量、请求头（Headers）和输出目录。未安装任何插件时会显示「尚未安装任何插件」。关于插件本身的开发与推荐配置写法，参见 [插件开发](/dev/plugins/)。

## 快速设置

以上常用选项同时暴露在「快速设置抽屉」里，可以在画廊、画册详情、画册列表、失败图片、自动配置与设置页的右上角直接打开，分为四组：

- **显示**：图片宽高比、双击图片（即「图片点击行为」）、列数、画廊布局、图片对齐方式。
- **下载**：同时运行任务数、最大并发下载量、下载间隔时间、网络失效重试次数、自动去重、导入推荐配置时默认启用定时、默认下载目录。
- **壁纸**（非 Web）：启用壁纸轮播、轮播间隔、轮播模式、壁纸显示方式、过渡效果、壁纸模式、Wallpaper Engine 目录。
- **应用**：开机启动（仅桌面）。

抽屉里的每一项都是完整设置页对应项的快捷入口，修改后会同步生效。

## 排障

**画册盘开启后在资源管理器里找不到虚拟盘** → 挂载可能失败。建议在设置中先关闭再重新开启画册盘。若仍失败，请确认已正确安装对应平台的底层驱动（Windows 需 Dokan，macOS 需 macFUSE）。

**切换壁纸模式后界面暂时无响应** → 切换期间相关控件会被禁用并显示等待光标，属于正常状态。若超过 30 秒仍未恢复，说明后端超时，刷新设置页后重试。

**过渡效果设为「滑动」或「缩放」但实际无过渡** → 原生模式下这两个选项会被自动修正为无过渡。如需使用过渡效果，请切换到窗口模式（Windows / macOS）或插件模式（KDE Plasma）。

**改了过渡效果后出现提示但效果未生效** → 当前轮播未启用。提示内容为「未启用轮播：过渡效果将在启用轮播后生效」，设置已保存，开启轮播后会自动应用。
