## 什么是画册？

画册是用于整理和分类图片的功能，类似于相册或收藏夹。你可以创建多个画册，把不同主题、来源或用途的图片分别整理到不同的画册中。

**画册与画廊的区别：**

- **画廊**：包含所有收集到的图片，按插件、时间等维度自动分类浏览
- **画册**：由你手动创建和管理，可以自由选择哪些图片加入画册

一张图片可以同时存在于多个画册中，画册只是对图片的引用，不会复制文件。

## 创建画册

在左侧菜单点开「画册」进入画册列表页，点击右上角的「新建画册」按钮，在弹出的对话框中输入名称后点击「创建」即可。

- 画册只有**名称一个字段**，没有描述、分类或封面字段。
- 建议名称控制在 **50 个字符以内**，对话框中长字符会被截断提示。
- 创建成功后会弹出「画册已创建」提示，列表会自动刷新并把新卡片插入进来。

### 创建子画册

画册支持嵌套成树状结构。进入任意画册详情页，点击页头的「新建子画册」按钮，同样输入名称即可。详情页顶部的面包屑会显示「画册 → 父画册 → 当前画册」的层级，方便回溯。

:::note
「收藏」画册和「隐藏」画册是两个特殊画册，不能在它们下面建子画册；隐藏画册也不显示「设为轮播」和「删除」。
:::

## 添加图片到画册

有多种方式把图片加入画册，入口统一复用同一个「加入画册」对话框：

- **从画廊添加**：在[画廊](/guide/gallery/)中选择图片（单选或多选），右键菜单选择「加入画册」，可选现有画册或现场新建。
- **从画册详情添加**：在当前画册里多选图片 → 右键「加入画册」，选择器会自动排除当前画册。
- **从任务详情添加（按图片）**：在任务详情页多选图片 → 右键「加入画册」。
- **从任务详情添加（整个任务）**（仅桌面）：任务详情页头的「加入画册」按钮，会把该任务下**全部**图片一次性入库。
- **导入时创建画册**：拖入文件或文件夹导入时，在确认对话框中勾选「为每个来源创建画册」，会自动按来源建画册并加入。

重复加入时会自动跳过已在画册中的图片，并提示「已跳过 {count} 张已在画册中的图片」；如果全部已在画册，提示「所选图片已全部在画册中」。

## 画册管理

画册列表页右键任一画册卡片，或进入详情页使用页头按钮，可以完成以下操作：

- **打开**：点击卡片进入详情页。
- **重命名**：进入详情页，**双击标题**进入编辑态（提示文案「双击改名」），回车确认，Esc 取消，失焦自动确认。
- **移动到**：把画册挂到另一个父画册下，或勾选「移动到根级」回到顶层。目标选择器会排除画册自身、它的后代以及「收藏」「隐藏」画册。
- **删除画册**：页头「删除画册」按钮。此操作**只删除画册和关联关系，不会删除图片文件**，图片在画廊和其他画册中保留。

:::note
「收藏」画册不能删除，尝试删除会被拦截。
:::

### 封面是自动生成的

画册封面由最近加入的几张图片**自动组合而成**（桌面显示前 3 张缩略图，Android 显示 1 张）。应用**不提供手动挑选封面**的入口，画册的预览会随着内容变化自动刷新。

### 隐藏画册

画册列表页右下角有一个悬浮按钮「进入隐藏画册」，用于存放你不想混在主列表里的内容。隐藏画册在主列表上的右键菜单被屏蔽，只能通过这个入口进入。

### 列表会自动刷新

加入、移除、删除、改名等操作完成后，画册预览、计数和详情列表会在 1 秒内自动同步，**无需手动点刷新**。

## 在画册中浏览

画册详情页顶部工具栏提供与画廊一致的浏览控制：

- **过滤**：「全部 / 仅设过壁纸 / 仅图片 / 仅视频」四选一；仅图片 / 仅视频会带计数显示。
- **排序**：
  - 在「全部 / 仅图片 / 仅视频」下可选「按时间升/降序」或「按加入顺序升/降序」。
  - 在「仅设过壁纸」下切换为「按设为壁纸时间升/降序」。
- **搜索**：工具栏右侧搜索框按图片的显示名做子串匹配。
- **每页条数**：100 / 500 / 1000 三档，与画廊共享同一项设置（见[设置](/guide/settings/)）。

当前画册有子画册时，详情页上方会显示「子画册」折叠区，带数量角标，点开可以像列表页一样管理。

## 从画册移除图片

在画册详情页选中图片 → 右键「从画册移除」，弹出确认对话框：

- **仅从画册移除**（默认）：只移除画册中的引用，图片文件和其他画册中的记录都保留。
- **同时删除图片**（仅桌面）：勾选对话框中的「同时删除图片（慎用）」后确认，会从画册移除的同时**永久删除电脑上的图片文件**，并从所有其他画册和画廊中移除。

:::caution
勾选「同时删除图片」后，操作**不可恢复**。当前正在用作壁纸的图片若被删除，对话框会给出额外提示。
:::

Android 端的对话框**不显示**「同时删除图片」复选框，只能选择仅从画册移除；如需彻底删除文件，请在桌面端操作。

### 上划删除（仅 Android）

在 Android 画册详情页，对图片上划（swipe）会直接执行「仅从画册移除」，不再弹出确认对话框。

## 画册与壁纸轮播

画册可以作为壁纸轮播源：

- **画册列表页**：右键画册卡片 → 「设为桌面轮播」。
- **画册详情页**：页头的「设为轮播壁纸」按钮。
- **设置页**：在「壁纸轮播」标签页中选择画册作为源。

如果壁纸轮播总开关没开，「设为轮播壁纸」会先帮你自动启用。空画册不能设为轮播源，会提示「画册为空：请先添加图片，再开启轮播」。

如果删除的画册正是当前轮播源，轮播会自动关闭并切回单张壁纸模式。

## 在虚拟磁盘中打开（仅桌面）

启用画册[虚拟磁盘](/guide/virtual-drive/)后（`albumDriveEnabled`），可以把画册当作本机文件夹访问：

- 画册列表页头提供「在虚拟磁盘中打开」入口，跳转到挂载根目录。
- 画册详情页头提供「在虚拟磁盘中打开该画册」入口，直接定位到当前画册目录。

默认挂载点为 Windows 上的 `K:\`，可以在设置中修改。Android、Web 版本和 Light 模式不提供虚拟磁盘功能。

## 平台差异

| 能力 | Windows | macOS | Linux | Android |
|---|---|---|---|---|
| 新建 / 重命名 / 删除画册 | OK | OK | OK | OK |
| 子画册 / 移动画册 | OK | OK | OK | OK |
| 加入画册（各入口） | OK | OK | OK | OK |
| 从画册仅移除 | OK | OK | OK | OK |
| 移除时同时删除图片文件 | OK | OK | OK | **不可用** |
| 打开画册虚拟磁盘 | 启用后 OK（Dokan） | 启用后 OK（macFUSE） | 启用后 OK（FUSE） | **不可用** |
| 设为壁纸轮播源 | OK | OK | OK | **不可用** |
| 图片导出到 Wallpaper Engine | OK（仅 Windows） | — | — | **不可用** |
| 分享图片 | — | — | — | OK |
| 上划删除 | — | — | — | OK（仅移除） |
