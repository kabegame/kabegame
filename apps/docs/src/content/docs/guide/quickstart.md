---
title: 快速上手
description: 五分钟内完成从安装插件到设置第一张壁纸的完整流程。
---

首次打开 Kabegame 会直接进入「画廊」页面，此时里面还没有任何图片，你会看到一个空态提示和一颗「开始收集」按钮。这篇文档沿着这个起点，带你走通「装插件 → 跑任务 → 看画廊 → 设为壁纸」的最短路径，全程大约 5 分钟。

每一步都有对应的深度文档，如果你想细致调参，可以顺着章节末尾的链接继续读。

## 第 1 步：安装一个源插件

Kabegame 本体不自带任何在线爬取源。「网络收集」要能选出可用源，必须先从插件商店装至少一个插件。

在左侧导航点击「源」进入插件管理页面。如果你还没有配置任何商店源，页面顶部会提示「当前没有任何商店源，可点击下方按钮切换到 GitHub 官方插件目录。」

:::tip
点击「打开 GitHub 官方插件商店」按钮，即可切换到官方目录。切换完成后商店标签页会列出所有可用插件，在任一卡片上点击「安装」；安装成功后按钮会变成「已安装」。
:::

如果你只想浏览本地已有的图片，可以跳过这一步，直接从第 2 步使用「本地」收集方式。

延伸阅读：[插件安装与管理](/guide/plugins-usage/)

## 第 2 步：开始收集图片

回到左侧导航的「画廊」页面。

:::tip
在画廊空态或右上角工具栏点击「开始收集」按钮，选择收集方式。
:::

目前提供两种：

- **本地**：从文件 / 文件夹 / zip 压缩包导入本地图片。桌面端也支持直接把文件拖入窗口。
- **网络**：使用你在第 1 步装好的插件从网络抓取图片。

选择「网络」后会打开抓取对话框。最少只需要在顶部「选择源」下拉里挑一个你已经装好的插件（或挑一个已保存的运行配置），底部「开始收集」按钮就会亮起，点击即可提交任务。

提交后任务会进入下载队列，图片开始陆续入库。你可以在「任务」页面看到它的实时进度。

延伸阅读：[任务与下载](/guide/tasks/)

## 第 3 步：在画廊里查看图片

任务下载过程中和结束后，画廊会自动刷新，无需手动重载。

画廊顶部工具栏的副标题会实时显示图片总数，例如「共 128 张图片」。当图片数超过一定数量时会自动启用分页，详细的浏览、预览、去重操作请参考画廊文档。

:::note
画廊空态默认文案里包含一段日文彩蛋，这是作者故意留下的，不代表界面语言异常。等第一张图片入库后这段提示就会被替换为正常的画廊视图。
:::

延伸阅读：[画廊](/guide/gallery/)

## 第 4 步：设为桌面壁纸（仅桌面）

在任意一张图片上点击右键，选择「抱到桌面上」，就能把它设为当前桌面壁纸。

如果你一次选中了多张图片再点「抱到桌面上」，Kabegame 会自动新建一个名为「桌面画册{n}」的画册，把所选图片放进去，并开启壁纸轮播。成功后会提示：「已开启轮播：画册「...」（N 张）」。

你也可以在画册上点击右键，选择「设为桌面轮播」，效果等同于把整个画册当作轮播源。

轮播间隔、是否跟随系统深色模式、Wallpaper Engine 导出等更多选项都集中在设置 →「壁纸设置」里。

延伸阅读：[壁纸设置](/guide/wallpaper/)

:::caution
Android 端目前不提供「单张图片直接设为壁纸」入口，右键菜单里的「抱到桌面上」会被平台守卫拦截。Android 用户的壁纸 / 轮播可行路径请参见 Android 专版说明（如有）或 [壁纸设置](/guide/wallpaper/) 文档中的平台差异说明。
:::

## 排障

**点了「网络」但源下拉是空的，「开始收集」按钮灰色不可点** → 还没安装任何源插件 → 回到第 1 步，前往「源」页安装至少一个插件后再重试。

**画廊一直没有更新，图片不出现** → 任务可能仍在下载中或已失败 → 前往[任务与下载](/guide/tasks/)页面确认任务状态；下载完成后画廊会自动刷新，无需手动操作。

**Android 上找不到拖拽导入** → Android 不支持桌面端的拖拽窗口 → 在「开始收集」里选「本地」，通过系统媒体选择器选图。
